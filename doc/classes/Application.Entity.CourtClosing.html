\Application\EntityCourtClosing

Entity class representing a date that the court is closed.

The CourtClosing is used for computing the difference between business dates. It could also be used for disabling certain dates on calendar controls or for date input validation. The Court is closed mostly for official holidays, but also, occasionally, for ad hoc events like blizzards and terror attacks. It is up to the user to keep the list of holidays up to date.
