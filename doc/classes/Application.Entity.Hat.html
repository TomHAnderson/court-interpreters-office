\Application\Entity

\Application\EntityHat

Entity class representing the "hat" (job description, title) a Person wears.

Examples: staff interpreter, contract interpreter, AUSA, USPO, defense attorney, etc. These should be set up at installation and rarely if ever changed thereafter.
